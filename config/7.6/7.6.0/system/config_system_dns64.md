# config system dns64

Configure DNS64.

## Syntax

```
config system dns64
    Description: Configure DNS64.
    set always-synthesize-aaaa-record [enable|disable]
    set dns64-prefix {ipv6-prefix}
    set status [enable|disable]
end
```

## Parameters

|                               | +-------------+--------------------------------------------------------+                         |

|                               | +=============+========================================================+                         |

|                               | +-------------+--------------------------------------------------------+                         |
