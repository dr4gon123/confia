# config ztna traffic-forward-proxy-reverse-service

Configure ZTNA traffic forward proxy reverse service.

## Syntax

```
config ztna traffic-forward-proxy-reverse-service
    Description: Configure ZTNA traffic forward proxy reverse service.
    config remote-servers
        Description: Connector Remote server
        edit <name>
            set address {string}
            set certificate {string}
            set health-check-interval {integer}
            set port {integer}
            set ssl-max-version [tls-1.1|tls-1.2|...]
            set status [enable|disable]
            set trusted-server-ca {string}
        next
    end
end
```
