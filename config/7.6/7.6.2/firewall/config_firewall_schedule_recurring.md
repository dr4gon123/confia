# config firewall schedule recurring

Recurring schedule configuration.

## Syntax

```
config firewall schedule recurring
    Description: Recurring schedule configuration.
    edit <name>
        set color {integer}
        set day {option1}, {option2}, ...
        set end {user}
        set fabric-object [enable|disable]
        set start {user}
        set uuid {uuid}
    next
end
```

## Parameters

+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| Parameter     | Description                       | Type               | Size               | Default                              |
+===============+===================================+====================+====================+======================================+
| color         | Color of icon on the GUI.         | integer            | Minimum value: 0   | 0                                    |
|               |                                   |                    | Maximum value: 32  |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| day           | One or more days of the week on   | option             | \-                 | none                                 |
|               | which the schedule is valid.      |                    |                    |                                      |
|               | Separate the names of the days    |                    |                    |                                      |
|               | with a space.                     |                    |                    |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
|               | +-------------+--------------------------------------------------------+                                           |
|               | | Option      | Description                                            |                                           |
|               | +=============+========================================================+                                           |
|               | | *sunday*    | Sunday.                                                |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *monday*    | Monday.                                                |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *tuesday*   | Tuesday.                                               |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *wednesday* | Wednesday.                                             |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *thursday*  | Thursday.                                              |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *friday*    | Friday.                                                |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *saturday*  | Saturday.                                              |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *none*      | None.                                                  |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| end           | Time of day to end the schedule,  | user               | Not Specified      |                                      |
|               | format hh:mm.                     |                    |                    |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| fabric-object | Security Fabric global object     | option             | \-                 | disable                              |
|               | setting.                          |                    |                    |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
|               | +-------------+--------------------------------------------------------+                                           |
|               | | Option      | Description                                            |                                           |
|               | +=============+========================================================+                                           |
|               | | *enable*    | Object is set as a security fabric-wide global object. |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
|               | | *disable*   | Object is local to this security fabric member.        |                                           |
|               | +-------------+--------------------------------------------------------+                                           |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| name          | Recurring schedule name.          | string             | Maximum length: 31 |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| start         | Time of day to start the          | user               | Not Specified      |                                      |
|               | schedule, format hh:mm.           |                    |                    |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
| uuid          | Universally Unique Identifier     | uuid               | Not Specified      | 00000000-0000-0000-0000-000000000000 |
|               | (UUID; automatically assigned but |                    |                    |                                      |
|               | can be manually reset).           |                    |                    |                                      |
+---------------+-----------------------------------+--------------------+--------------------+--------------------------------------+
