# config system sov-sase

Configure Sovereign SASE.

## Syntax

```
config system sov-sase
    Description: Configure Sovereign SASE.
    set status [enable|disable]
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| status    | Enable/disable Sovereign SASE.    | option             | \-                 | disable            |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *enable*    | Enable Sovereign SASE.                                 |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *disable*   | Disable Sovereign SASE.                                |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+--------------------------------------------------------------------------------------------------+
