# config system sdn-vpn

Configure public cloud VPN service.

## Syntax

```
config system sdn-vpn
    Description: Configure public cloud VPN service.
    edit <name>
        set bgp-as {integer}
        set cgw-gateway {ipv4-address-any}
        set cgw-name {string}
        set code {integer}
        set internal-interface {string}
        set local-cidr {ipv4-classnet}
        set nat-traversal [disable|enable]
        set psksecret {password-3}
        set remote-cidr {ipv4-classnet}
        set remote-type [vgw|tgw]
        set routing-type [static|dynamic]
        set sdn {string}
        set status {integer}
        set subnet-id {string}
        set tgw-id {string}
        set tunnel-interface {string}
        set type {integer}
        set vgw-id {string}
    next
end
```
