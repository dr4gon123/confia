# config system fips-cc

Configure FIPS-CC mode.

## Syntax

```
config system fips-cc
    Description: Configure FIPS-CC mode.
    set key-generation-self-test [enable|disable]
    set self-test-period {integer}
    set status [enable|disable]
end
```

## Parameters

+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter                | Description                       | Type               | Size               | Default            |
+==========================+===================================+====================+====================+====================+
| key-generation-self-test | Enable/disable self tests after   | option             | \-                 | disable            |
|                          | key generation.                   |                    |                    |                    |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | Option      | Description                                            |                         |
|                          | +=============+========================================================+                         |
|                          | | *enable*    | Enable self tests after key generation.                |                         |
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | *disable*   | Disable self tests after key generation.               |                         |
|                          | +-------------+--------------------------------------------------------+                         |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| self-test-period         | Self test period.                 | integer            | Minimum value: 5   | 1440               |
|                          |                                   |                    | Maximum value:     |                    |

| status \*                | Enable/disable ciphers for FIPS   | option             | \-                 | disable            |

|                          | +-------------+--------------------------------------------------------+                         |
|                          | | Option      | Description                                            |                         |
|                          | +=============+========================================================+                         |
