# config webfilter ftgd-local-risk

Configure FortiGuard Web Filter local risk score.

## Syntax

```
config webfilter ftgd-local-risk
    Description: Configure FortiGuard Web Filter local risk score.
    edit <url>
        set comment {var-string}
        set risk-score {integer}
        set status [enable|disable]
    next
end
```

## Parameters

+------------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter  | Description                       | Type               | Size               | Default            |
+============+===================================+====================+====================+====================+
| comment    | Comment.                          | var-string         | Maximum length:    |                    |
|            |                                   |                    | 255                |                    |
+------------+-----------------------------------+--------------------+--------------------+--------------------+
| risk-score | Local risk score.                 | integer            | Minimum value: 0   | 0                  |
|            |                                   |                    | Maximum value: 100 |                    |
+------------+-----------------------------------+--------------------+--------------------+--------------------+
| status     | Enable/disable local risk score.  | option             | \-                 | enable             |
+------------+-----------------------------------+--------------------+--------------------+--------------------+
|            | +-------------+--------------------------------------------------------+                         |
|            | | Option      | Description                                            |                         |
|            | +=============+========================================================+                         |
|            | | *enable*    | Enable local risk level.                               |                         |
|            | +-------------+--------------------------------------------------------+                         |
|            | | *disable*   | Disable local risk level.                              |                         |
|            | +-------------+--------------------------------------------------------+                         |
+------------+-----------------------------------+--------------------+--------------------+--------------------+
| url        | URL to rate locally.              | string             | Maximum length:    |                    |
|            |                                   |                    | 511                |                    |
+------------+-----------------------------------+--------------------+--------------------+--------------------+
