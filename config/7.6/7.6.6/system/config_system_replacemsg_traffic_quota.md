# config system replacemsg traffic-quota

Replacement messages. Read-only.

## Syntax

```
config system replacemsg traffic-quota
    Description: Replacement messages. Read-only.
    edit <msg-type>
        set buffer {var-string}
        set format [none|text|...]
        set header [none|http|...]
    next
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| buffer    | Message string.                   | var-string         | Maximum length:    |                    |
|           |                                   |                    | 32768              |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| format    | Format flag.                      | option             | \-                 | none               |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *none*      | No format type.                                        |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *text*      | Text format.                                           |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *html*      | HTML format.                                           |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| header    | Header flag.                      | option             | \-                 | none               |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *none*      | No header type.                                        |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *http*      | HTTP                                                   |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *8bit*      | 8 bit.                                                 |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| msg-type  | Message type.                     | string             | Maximum length: 28 |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
