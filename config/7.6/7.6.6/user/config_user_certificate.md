# config user certificate

Configure certificate users.

## Syntax

```
config user certificate
    Description: Configure certificate users.
    edit <name>
        set common-name {string}
        set id {integer}
        set issuer {string}
        set status [enable|disable]
        set type [single-certificate|trusted-issuer]
    next
end
```

## Parameters

+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| Parameter   | Description                       | Type                  | Size                  | Default               |
+=============+===================================+=======================+=======================+=======================+
| common-name | Certificate common name.          | string                | Maximum length: 64    |                       |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| id          | User ID. Read-only.               | integer               | Minimum value: 0      | 0                     |
|             |                                   |                       | Maximum value:        |                       |
|             |                                   |                       | 4294967295            |                       |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| issuer      | CA certificate used for client    | string                | Maximum length: 79    |                       |
|             | certificate verification.         |                       |                       |                       |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| name        | User name.                        | string                | Maximum length: 64    |                       |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| status      | Enable/disable allowing the       | option                | \-                    | enable                |
|             | certificate user to authenticate  |                       |                       |                       |
|             | with the FortiGate unit.          |                       |                       |                       |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
|             | +-------------+--------------------------------------------------------+                                  |
|             | | Option      | Description                                            |                                  |
|             | +=============+========================================================+                                  |
|             | | *enable*    | Enable user.                                           |                                  |
|             | +-------------+--------------------------------------------------------+                                  |
|             | | *disable*   | Disable user.                                          |                                  |
|             | +-------------+--------------------------------------------------------+                                  |
+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| type        | Type of certificate               | option                | \-                    | single-certificate    |

+-------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
|             | +----------------------+--------------------------------------------------------+                         |
|             | | Option               | Description                                            |                         |
|             | +======================+========================================================+                         |
|             | | *single-certificate* | Single certificate.                                    |                         |
|             | +----------------------+--------------------------------------------------------+                         |

|             | +----------------------+--------------------------------------------------------+                         |
