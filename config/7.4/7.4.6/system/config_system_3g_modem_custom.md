# config system 3g-modem custom

3G MODEM custom.

## Syntax

```
config system 3g-modem custom
    Description: 3G MODEM custom.
    edit <id>
        set class-id {user}
        set init-string {string}
        set model {string}
        set modeswitch-string {string}
        set product-id {user}
        set vendor {string}
        set vendor-id {user}
    next
end
```
