# config system console

Configure console.

## Syntax

```
config system console
    Description: Configure console.
    set baudrate [9600|19200|...]
    set fortiexplorer [enable|disable]
    set login [enable|disable]
    set mode [batch|line]
    set output [standard|more]
end
```
