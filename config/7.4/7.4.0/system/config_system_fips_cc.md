# config system fips-cc

Configure FIPS-CC mode.

## Syntax

```
config system fips-cc
    Description: Configure FIPS-CC mode.
    set entropy-token [enable|disable|...]
    set key-generation-self-test [enable|disable]
    set self-test-period {integer}
    set status [enable|disable]
end
```

## Parameters

+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter                | Description                       | Type               | Size               | Default            |
+==========================+===================================+====================+====================+====================+
| entropy-token            | Enable/disable/dynamic entropy    | option             | \-                 | enable             |
|                          | token.                            |                    |                    |                    |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | Option      | Description                                            |                         |
|                          | +=============+========================================================+                         |
|                          | | *enable*    | Enable entropy token to be present during boot         |                         |
|                          | |             | process.                                               |                         |
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | *disable*   | Disable entropy token to be present during boot        |                         |
|                          | |             | process.                                               |                         |
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | *dynamic*   | Dynamic detect entropy token to be present during boot |                         |
|                          | |             | process.                                               |                         |
|                          | +-------------+--------------------------------------------------------+                         |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| key-generation-self-test | Enable/disable self tests after   | option             | \-                 | disable            |
|                          | key generation.                   |                    |                    |                    |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | Option      | Description                                            |                         |
|                          | +=============+========================================================+                         |
|                          | | *enable*    | Enable self tests after key generation.                |                         |
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | *disable*   | Disable self tests after key generation.               |                         |
|                          | +-------------+--------------------------------------------------------+                         |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| self-test-period         | Self test period.                 | integer            | Minimum value: 1   | 1440               |
|                          |                                   |                    | Maximum value:     |                    |
|                          |                                   |                    | 1440               |                    |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
| status                   | Enable/disable ciphers for FIPS   | option             | \-                 | disable            |
|                          | mode of operation.                |                    |                    |                    |
+--------------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | Option      | Description                                            |                         |
|                          | +=============+========================================================+                         |
|                          | | *enable*    | Enable FIPS-CC mode.                                   |                         |
|                          | +-------------+--------------------------------------------------------+                         |
|                          | | *disable*   | Disable FIPS-CC mode.                                  |                         |
|                          | +-------------+--------------------------------------------------------+                         |
