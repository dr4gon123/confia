# config user fortitoken

Configure FortiToken.

## Syntax

```
config user fortitoken
    Description: Configure FortiToken.
    edit <serial-number>
        set activation-code {string}
        set activation-expire {integer}
        set comments {var-string}
        set license {string}
        set os-ver {string}
        set reg-id {string}
        set seed {string}
        set status [active|lock]
    next
end
```

## Parameters

+-------------------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter         | Description                       | Type               | Size               | Default            |
+===================+===================================+====================+====================+====================+
| activation-code   | Mobile token user                 | string             | Maximum length: 32 |                    |
|                   | activation-code.                  |                    |                    |                    |
+-------------------+-----------------------------------+--------------------+--------------------+--------------------+
| activation-expire | Mobile token user activation-code | integer            | Minimum value: 0   | 0                  |
|                   | expire time.                      |                    | Maximum value:     |                    |
|                   |                                   |                    | 4294967295         |                    |
+-------------------+-----------------------------------+--------------------+--------------------+--------------------+
