# config ips view-map

Configure IPS view-map.

## Syntax

```
config ips view-map
    Description: Configure IPS view-map.
    edit <id>
        set id-policy-id {integer}
        set policy-id {integer}
        set vdom-id {integer}
        set which [firewall|interface|...]
    next
end
```
