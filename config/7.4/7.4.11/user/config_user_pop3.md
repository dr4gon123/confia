# config user pop3

POP3 server entry configuration.

## Syntax

```
config user pop3
    Description: POP3 server entry configuration.
    edit <name>
        set port {integer}
        set secure [none|starttls|...]
        set server {string}
        set ssl-min-proto-version [default|SSLv3|...]
    next
end
```

## Parameters

+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter             | Description                       | Type               | Size               | Default            |
+=======================+===================================+====================+====================+====================+
| name                  | POP3 server entry name.           | string             | Maximum length: 35 |                    |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
| port                  | POP3 service port number.         | integer            | Minimum value: 0   | 0                  |
|                       |                                   |                    | Maximum value:     |                    |
|                       |                                   |                    | 65535              |                    |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
| secure                | SSL connection.                   | option             | \-                 | starttls           |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | Option      | Description                                            |                         |
|                       | +=============+========================================================+                         |
|                       | | *none*      | None.                                                  |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *starttls*  | Use StartTLS.                                          |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *pop3s*     | Use POP3 over SSL.                                     |                         |
|                       | +-------------+--------------------------------------------------------+                         |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
| server                | Server domain name or IP address. | string             | Maximum length: 63 |                    |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
| ssl-min-proto-version | Minimum supported protocol        | option             | \-                 | default            |
|                       | version for SSL/TLS connections   |                    |                    |                    |
|                       | (default is to follow system      |                    |                    |                    |
|                       | global setting).                  |                    |                    |                    |
+-----------------------+-----------------------------------+--------------------+--------------------+--------------------+
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | Option      | Description                                            |                         |
|                       | +=============+========================================================+                         |
|                       | | *default*   | Follow system global setting.                          |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *SSLv3*     | SSLv3.                                                 |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *TLSv1*     | TLSv1.                                                 |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *TLSv1-1*   | TLSv1.1.                                               |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *TLSv1-2*   | TLSv1.2.                                               |                         |
|                       | +-------------+--------------------------------------------------------+                         |
|                       | | *TLSv1-3*   | TLSv1.3.                                               |                         |
|                       | +-------------+--------------------------------------------------------+                         |
+-----------------------+--------------------------------------------------------------------------------------------------+
