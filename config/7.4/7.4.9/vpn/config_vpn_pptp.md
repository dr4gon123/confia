# config vpn pptp

Configure PPTP.

## Syntax

```
config vpn pptp
    Description: Configure PPTP.
    set eip {ipv4-address}
    set ip-mode [range|usrgrp]
    set local-ip {ipv4-address}
    set sip {ipv4-address}
    set status [enable|disable]
    set usrgrp {string}
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| eip       | End IP.                           | ipv4-address       | Not Specified      | 0.0.0.0            |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| ip-mode   | IP assignment mode for PPTP       | option             | \-                 | range              |
|           | client.                           |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *range*     | PPTP client IP from manual config (range from sip to   |                         |
|           | |             | eip).                                                  |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *usrgrp*    | PPTP client IP from user-group defined server.         |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| local-ip  | Local IP to be used for peer\'s   | ipv4-address       | Not Specified      | 0.0.0.0            |
|           | remote IP.                        |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| sip       | Start IP.                         | ipv4-address       | Not Specified      | 0.0.0.0            |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| status    | Enable/disable FortiGate as a     | option             | \-                 | disable            |
|           | PPTP gateway.                     |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *enable*    | Enable setting.                                        |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *disable*   | Disable setting.                                       |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| usrgrp    | User group.                       | string             | Maximum length: 35 |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
