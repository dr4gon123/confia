# config monitoring np6-ipsec-engine

Configure NP6 IPsec engine status monitoring.

## Syntax

```
config monitoring np6-ipsec-engine
    Description: Configure NP6 IPsec engine status monitoring.
    set interval {integer}
    set status [enable|disable]
    set threshold {user}
end
```
