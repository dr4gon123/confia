# config wireless-controller wtp-group

Configure WTP groups.

## Syntax

```
config wireless-controller wtp-group
    Description: Configure WTP groups.
    edit <name>
        set ble-major-id {integer}
        set platform-type [AP-11N|C24JE|...]
        set wtps <wtp-id1>, <wtp-id2>, ...
    next
end
```
