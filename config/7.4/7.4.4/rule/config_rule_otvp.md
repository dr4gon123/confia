# config rule otvp

Show OT patch signatures.

## Syntax

```
config rule otvp
    Description: Show OT patch signatures.
    edit <name>
        set action [pass|block]
        set application {user}
        set date {integer}
        set group {string}
        set location {user}
        set log [disable|enable]
        set log-packet [disable|enable]
        config metadata
            Description: Meta data.
            edit <id>
                set metaid {integer}
                set valueid {integer}
            next
        end
        set os {user}
        set rev {integer}
        set rule-id {integer}
        set service {user}
        set severity {user}
    next
end
```
