# config log webtrends setting

Settings for WebTrends.

## Syntax

```
config log webtrends setting
    Description: Settings for WebTrends.
    set server {string}
    set status [enable|disable]
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| server    | Address of the remote WebTrends   | string             | Maximum length: 63 |                    |
|           | server.                           |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| status    | Enable/disable logging to         | option             | \-                 | disable            |
|           | WebTrends.                        |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *enable*    | Enable logging to WebTrends.                           |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *disable*   | Disble logging to WebTrends.                           |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+--------------------------------------------------------------------------------------------------+
