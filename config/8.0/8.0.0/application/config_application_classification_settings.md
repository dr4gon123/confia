# config application classification-settings

Configure app classification setting.

## Syntax

```
config application classification-settings
    Description: Configure app classification setting.
    set default-app-classification [unclassified|unsanctioned|...]
end
```

## Parameters

+----------------------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| Parameter                  | Description                       | Type                  | Size                  | Default               |
+============================+===================================+=======================+=======================+=======================+
| default-app-classification | Default application               | option                | \-                    | unclassified          |
|                            | classification.                   |                       |                       |                       |
+----------------------------+-----------------------------------+-----------------------+-----------------------+-----------------------+
|                            | +----------------+--------------------------------------------------------+                               |
|                            | | Option         | Description                                            |                               |
|                            | +================+========================================================+                               |
|                            | | *unclassified* | Set unclassified.                                      |                               |
|                            | +----------------+--------------------------------------------------------+                               |
|                            | | *unsanctioned* | Set unsanctioned.                                      |                               |
|                            | +----------------+--------------------------------------------------------+                               |
|                            | | *sanctioned*   | Set sanctioned.                                        |                               |
|                            | +----------------+--------------------------------------------------------+                               |
+----------------------------+-----------------------------------------------------------------------------------------------------------+
