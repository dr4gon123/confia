# config application unsanctioned-apps

Configure unsanctioned applications.

## Syntax

```
config application unsanctioned-apps
    Description: Configure unsanctioned applications.
    edit <id>
        set app {integer}
        set category {user}
        set status [unsanctioned|sanctioned]
        set type [app|category]
    next
end
```
