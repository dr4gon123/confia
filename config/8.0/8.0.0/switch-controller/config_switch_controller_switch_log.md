# config switch-controller switch-log

Configure FortiSwitch logging (logs are transferred to and inserted into FortiGate event log).

## Syntax

```
config switch-controller switch-log
    Description: Configure FortiSwitch logging (logs are transferred to and inserted into FortiGate event log).
    set severity [emergency|alert|...]
    set status [enable|disable]
end
```

## Parameters

+-----------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| Parameter | Description                       | Type                  | Size                  | Default               |
+===========+===================================+=======================+=======================+=======================+
| severity  | Severity of FortiSwitch logs that | option                | \-                    | notification          |
|           | are added to the FortiGate event  |                       |                       |                       |
|           | log.                              |                       |                       |                       |
+-----------+-----------------------------------+-----------------------+-----------------------+-----------------------+
|           | +----------------+--------------------------------------------------------+                               |
|           | | Option         | Description                                            |                               |
|           | +================+========================================================+                               |
|           | | *emergency*    | Emergency level.                                       |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *alert*        | Alert level.                                           |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *critical*     | Critical level.                                        |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *error*        | Error level.                                           |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *warning*      | Warning level.                                         |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *notification* | Notification level.                                    |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *information*  | Information level.                                     |                               |
|           | +----------------+--------------------------------------------------------+                               |
|           | | *debug*        | Debug level.                                           |                               |
|           | +----------------+--------------------------------------------------------+                               |
+-----------+-----------------------------------+-----------------------+-----------------------+-----------------------+
| status    | Enable/disable adding FortiSwitch | option                | \-                    | enable                |
|           | logs to FortiGate event log.      |                       |                       |                       |
+-----------+-----------------------------------+-----------------------+-----------------------+-----------------------+
|           | +-------------+--------------------------------------------------------+                                  |
|           | | Option      | Description                                            |                                  |
|           | +=============+========================================================+                                  |
|           | | *enable*    | Add FortiSwitch logs to FortiGate event log.           |                                  |
|           | +-------------+--------------------------------------------------------+                                  |
|           | | *disable*   | Do not add FortiSwitch logs to FortiGate event log.    |                                  |
|           | +-------------+--------------------------------------------------------+                                  |
+-----------+-----------------------------------------------------------------------------------------------------------+
