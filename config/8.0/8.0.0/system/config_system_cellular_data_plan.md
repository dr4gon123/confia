# config system cellular-data-plan

Configure cellular data plan.

## Syntax

```
config system cellular-data-plan
    Description: Configure cellular data plan.
    edit <name>
        set apn {string}
        set auth-type [none|pap|...]
        set password {password}
        set pdp-type [ipv4|ipv6|...]
        set username {string}
    next
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| apn       | Login APN string for PDP-IP       | string             | Maximum length:    |                    |
|           | packet data calls.                |                    | 127                |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| auth-type | Authentication type for PDP-IP    | option             | \-                 | none               |
|           | packet data calls.                |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *none*      | Username and password not required                     |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *pap*       | Use PAP authentication.                                |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *chap*      | Use CHAP authentication.                               |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| name      | Data plan name.                   | string             | Maximum length: 35 |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| password  | Authentication password for       | password           | Not Specified      |                    |
|           | PDP-IP packet data calls.         |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| pdp-type  | Packet Data Protocol (PDP)        | option             | \-                 | ipv4v6             |
|           | context type.                     |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *ipv4*      | IPv4 only                                              |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *ipv6*      | IPv6 only.                                             |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *ipv4v6*    | Dual stack for IPv4 and IPv6.                          |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| username  | Authentication username for       | string             | Maximum length: 63 |                    |
|           | PDP-IP packet data calls.         |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
