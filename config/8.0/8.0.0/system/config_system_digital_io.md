# config system digital-io

Configure digital-io.

## Syntax

```
config system digital-io
    Description: Configure digital-io.
    set input1-detection-mode [default|voltage]
    set input2-detection-mode [default|voltage]
    set output-keep-last-state [enable|disable]
end
```
