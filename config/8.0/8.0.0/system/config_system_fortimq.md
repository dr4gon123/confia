# config system fortimq

Configure FortiMQ settings.

## Syntax

```
config system fortimq
    Description: Configure FortiMQ settings.
    set ocsp-check [enable|disable]
    set publish-metadata [enable|disable]
    set status [enable|disable]
end
```
