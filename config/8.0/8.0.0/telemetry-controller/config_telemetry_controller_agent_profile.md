# config telemetry-controller agent-profile

Configure FortiTelemetry agent profiles.

## Syntax

```
config telemetry-controller agent-profile
    Description: Configure FortiTelemetry agent profiles.
    edit <name>
        set comment {var-string}
        set model [ftl-100g|windows|...]
    next
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| comment   | Comment.                          | var-string         | Maximum length:    |                    |
|           |                                   |                    | 255                |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| model     | Model.                            | option             | \-                 | ftl-100g           |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *ftl-100g*  | FTL-100G.                                              |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *windows*   | Microsoft Windows agent.                               |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *macos*     | Apple macOS agent.                                     |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| name      | Name of the agent profile.        | string             | Maximum length: 35 |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
