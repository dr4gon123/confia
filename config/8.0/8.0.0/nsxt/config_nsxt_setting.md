# config nsxt setting

Configure NSX-T setting.

## Syntax

```
config nsxt setting
    Description: Configure NSX-T setting.
    set liveness [enable|disable]
    set service {string}
end
```

## Parameters

+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| Parameter | Description                       | Type               | Size               | Default            |
+===========+===================================+====================+====================+====================+
| liveness  | Enable/disable liveness detection | option             | \-                 | disable            |
|           | packet forwarding.                |                    |                    |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
|           | +-------------+--------------------------------------------------------+                         |
|           | | Option      | Description                                            |                         |
|           | +=============+========================================================+                         |
|           | | *enable*    | Enable liveness detection packet forwarding.           |                         |
|           | +-------------+--------------------------------------------------------+                         |
|           | | *disable*   | Disable liveness detection packet forwarding.          |                         |
|           | +-------------+--------------------------------------------------------+                         |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
| service   | Service name.                     | string             | Maximum length: 63 |                    |
+-----------+-----------------------------------+--------------------+--------------------+--------------------+
